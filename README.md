# 🎵 Nocturne - Music Player

Ứng dụng nghe nhạc hiện đại được tối ưu hóa cho trải nghiệm người dùng tuyệt vời.

## ✨ Tính năng chính

- 🎶 **Trình phát nhạc** - Đầy đủ bộ điều khiển: phát, tạm dừng, bỏ qua track.
- 📱 **Thiết kế Responsive** - Hoạt động mượt mà trên cả máy tính và thiết bị di động.
- 🔐 **Xác thực người dùng** - Đăng nhập, đăng ký và quản lý hồ sơ cá nhân.
- 📊 **Nhạc Trending** - Khám phá những bài hát đang phổ biến nhất.
- ⏰ **Lịch sử nghe nhạc** - Theo dõi các bài hát bạn đã nghe gần đây.
- 🌙 **Giao diện tối (Dark Mode)** - Thiết kế hiện đại với màu sắc Neon bắt mắt.
- 🔄 **Tích hợp API thực tế** - Kết nối trực tiếp với hệ thống backend để quản lý dữ liệu.
- ⚡ **Tốc độ vượt trội** - Sử dụng Vite để khởi động và cập nhật tức thì.

## 🛠 Công nghệ sử dụng

- **React 19** - Thư viện UI mạnh mẽ và hiện đại.
- **Vite 7** - Công cụ build và dev server tốc độ cao.
- **Tailwind CSS 3** - Framework CSS giúp tùy chỉnh giao diện linh hoạt.
- **Material Symbols** - Bộ icon hiện đại từ Google.
- **Context API** - Quản lý trạng thái ứng dụng hiệu quả.
